agel-form | 使用文档

基于 element-ui form 的二次封装加强组件

起步 →

极简的思想

面向对象, 参数集中处理, 以一个 form 对象来完成所有的操作

高度的灵活

支持 element-ui form 所有组件，属性，方法，插槽，灵活搭配自定义/第三方组件，

快捷的使用

数据配置，全局配置，集成了更方便好用的扩展功能

# 如此简单

<template>
  <agel-form v-model="form"></agel-form>
</template>
 
<script>
export default {
  data() {
    return {
      form: {},
    };
  },
};
</script>
